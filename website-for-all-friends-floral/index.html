Check out the website

Organizing styles & components

Careful planning made rapid prototyping much easier. I worked with Sabrina, the owner of All Friends Floral, to choose typography and color palettes, as well as gather general inspiration. I used Sketch to organize all UI and typography documentation for easy reference. I also converted all reusable UI patterns into symbols.

Color and typography artboards in Sketch

Making pretty graphics

Blending Sabrina’s aesthetic and the tactile nature of flowers, I designed graphics with transparent edges and overlapping features. I used the shape of her logo as a pattern for brand consistency. I created all the graphics in Photoshop and Illustrator.

Collage style graphics of Sabrina and floral arrangements

Building the thing

After creating each page design in Sketch and prototyping in Invision, I built the website in a local environment with MAMP and Sublime Text 3. I implemented a basic Bootstrap Wordpress theme and custom CSS, HTML, and jQuery. Some of my custom coded features include:

  1. Instagram feed using the instafeed.js plugin to call Instagram’s API.
  2. Image loading with using the waitForImages.js plugin to trigger CSS transitions.
  3. Responsive typography with CSS.

Some flexbox fun to build the Instagram feed
